 - Flutter 2.5.2
 - Dart 2.14.3
 - Android Studio Arctic Fox 2020.3.1 Patch 2
 - Dart Plugin 202.8531
 - Flutter Plugin 61.0.2
 - Android SDK version 31.0.0
 - Xcode 12.5.1
